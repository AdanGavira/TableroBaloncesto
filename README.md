# TableroBaloncesto

TableroBaloncesto es un proyecto sobre un marcador de un partido de baloncesto. 

## Características

Actualización en tiempo real del marcador.
Control de periodos y tiempos.
Interfaz intuitiva y fácil de usar.

## Uso

1. Ejecuta la aplicación según las instrucciones de instalación.
2. Accede a la interfaz gráfica o terminal.
3. Configura los equipos participantes y utiliza los controles para llevar el marcador.
